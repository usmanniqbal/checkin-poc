# Install CLI
```
$ npm install -g @ionic/cli
```

# Install Packages

```
npm i
```

# To Run in web browser 
```
ionic start
```

# To run in device/simulator
## Sync Packages
```
ionic capacitor sync
```
## Open in Native IDE
```
ionic capacitor open ios/android
```
## Run with live reload
```
ionic capacitor run ios/android -l
```
